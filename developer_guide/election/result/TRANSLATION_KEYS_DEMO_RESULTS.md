## Implementation Instructions

### 1. Add Keys to i18n Files

Place these keys in the appropriate locale JSON files:

```
resources/js/locales/pages/Demo/Result/en.json
resources/js/locales/pages/Demo/Result/de.json
resources/js/locales/pages/Demo/Result/np.json
```

### 2. Update Vue Components

Replace hardcoded text with `$t()` function:

**Before:**
```vue
<h1>Demo Election Results</h1>
```

**After:**
```vue
<h1>{{ $t('pages.demo-result.title') }}</h1>
```

### 3. Component Examples

**Index.vue:**
```vue
<h1 class="text-3xl font-bold">
  {{ $t('pages.demo-result.title') }}
</h1>

<p class="text-gray-600">
  {{ $t('pages.demo-result.subtitle') }}
</p>

<button @click="downloadPDF">
  {{ $t('pages.demo-result.actions.download-pdf') }}
</button>
```

**ModeIndicator.vue:**
```vue
<p class="font-semibold">{{ title }}</p>
<p class="text-sm opacity-90">{{ description }}</p>

<script setup>
const title = computed(() =>
  isGlobal.value
    ? $t('components.mode-indicator.global-title')
    : $t('components.mode-indicator.organisation-title')
);
</script>
```

### 4. Testing Translations

```bash
# Test English
php artisan tinker
>>> app()->setLocale('en')

# Test German
>>> app()->setLocale('de')

# Test Nepali
>>> app()->setLocale('np')
```
