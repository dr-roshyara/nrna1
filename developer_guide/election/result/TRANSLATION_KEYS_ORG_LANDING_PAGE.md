## Implementation Instructions

### Step 1: Locate Existing Files

Navigate to:
```
resources/js/locales/pages/Organizations/Show/
```

Expected files:
```
├── en.json
├── de.json
└── np.json
```

### Step 2: Add Keys to Existing File

**For each file (en.json, de.json, np.json):**

1. Open the file
2. Find the closing brace `}`
3. Add a comma before it
4. Add the keys from above (without the outer `{}`)
5. Keep same indentation as existing keys

**Example (en.json):**

```json
{
  "pages.organisation-show.actions.title": "Quick Actions",

  // ... existing keys ...

  "pages.organisation-show.demo-results.title": "Demo Election Results",  ← ADD THESE

  // ... rest of keys ...
}
```

The `DemoResultsSection.vue` component uses:


## Key organisation

**Keys are organized hierarchically:**

```
pages                     ← Top level (page-specific)
  .organisation-show      ← Page name
    .demo-results         ← Component/section
      .title              ← Specific key
      .description
      .mode-1
      .mode-2
      .feature-*          ← Feature list items
      .button-*           ← Action buttons
      .info-*             ← Information box
```
