## For New Developers

### Setup Instructions
1. Copy `.env.example` to `.env`
   ```bash
   cp .env.example .env
   ```

2. Update database credentials
   ```env
   DB_HOST=your-host
   DB_DATABASE=your-database
   DB_USERNAME=your-user
   DB_PASSWORD=your-password
   ```

3. Update mail settings (or use Mailhog for local dev)
   ```env
   MAIL_HOST=mailhog
   MAIL_PORT=1025
   ```

4. Generate application key
   ```bash
   php artisan key:generate
   ```

5. Run migrations
   ```bash
   php artisan migrate
   ```
